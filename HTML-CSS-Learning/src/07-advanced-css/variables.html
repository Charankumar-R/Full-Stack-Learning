Understanding CSS Variables

What are CSS Variables?

CSS Variables, also known as custom properties, allow you to store values that you can reuse throughout your CSS. They are defined using the --variable-name syntax.

Defining CSS Variables

CSS variables are defined within a selector using the var(--variable-name) syntax. For example:

:root {
    --main-color: #3498db;
    --padding: 16px;
}

Using CSS Variables

Once defined, you can use these variables in your styles:

body {
    background-color: var(--main-color);
    padding: var(--padding);
}

Benefits of CSS Variables

  • Easy to maintain and update styles.
  • Dynamic changes with JavaScript.
  • Scoped to the element they are defined in.
